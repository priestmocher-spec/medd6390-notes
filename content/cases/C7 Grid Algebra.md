# C7 Grid Algebra

**文件：** example（gp）/Grid Algebra.pdf（12 页）
**来源：** Dave Hewitt, Digital Experiences in Mathematics Education (2016) + LUMEN KS3 资源
**数学主题：** 代数表达式、展开与提取公因式、等价表达式、方程求解
**适用年级：** Secondary (11-14 岁)

## 网格结构

网格有行和列。核心空间隐喻：
- **右移 = +1，左移 = -1**
- **下移 = ×行号比，上移 = ÷行号比**

## 核心活动

### 活动 1：路径与表达式
从第1行的 n 出发，向右3步到 n+3，再向下到第2行变成 2(n+3)。不同路径 → 不同表达式，但同一终点 → 表达式等价。

### 活动 2：展开与因式分解
- 路径 A："先加3再乘2" → 2(f+3)
- 路径 B："先乘2再加6" → 2f+6
- 到达同一格子 → 2(f+3) = 2f+6
- 提供展开/因式分解的视觉和动态理解

### 活动 3：等价表达式（软件 Task 3 & 4）
给定起点和彩色目标格，输入三条不同路径（等价表达式）到达目标

### 活动 4：练习 Worksheet（20 题）
从 2x+6 到 17x-85+34h-119，难度递增

### 活动 5：写表达式比赛
对数字 4,9,17,35,61 各写 5 种不同表达式，约束递增

## 设计原则 (Hewitt, 2016)

三个理论框架：
1. 基于 Gattegno (1988)：代数是"运算之上的运算"
2. 利用学生的 deep-rooted knowings（日常运算直觉）
3. 通过软件设计 address 已知的代数误概念

**软件限制作为教学手段**：Grid Algebra 中无法写出某些等价表达式，促使学生思考软件表达与标准代数表达的差异。
