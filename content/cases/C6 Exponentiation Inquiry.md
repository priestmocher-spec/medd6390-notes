## Part A：指数的基本理解

### NCETM Example 7（同底变式）
- (2^3)^2 = 2^a, (4^b)^2 = 4^8, (8^2)^(1/3) = 8^c, (16^3)^d = 16^(3/4)

### NCETM Example 12（换底变式）
- (2^3)^2 = 8^a, (4^b)^2 = 2^16, (8^2)^(1/3) = 4^c
- 与 Example 7 的区别：等号两边底数不同，需要在不同底之间转换
- **这是刻意的 variation design**

### NCETM Example 11（开放探索）
- 找满足 (7^m)^n = 7 的 m, n 值对。"再找一对...再找一对..."

## Part C：对数的引入

- **Jacobs (1994) 教科书的对数表活动**：二进制对数表，发现"乘法 = 对数的加法"
- **对数标尺活动**：时间单位转换（秒到世纪），引出对数标尺的必要性
- **Benjamin Franklin 遗赠**：$2,550 每年4%复利200年，用对数值的线性增长预测

## Part D：指数增长的现实应用

- **Duckweed（浮萍）**：面积每天翻倍 a=2^t，引入 l(a) 符号发现对数性质
- **骰子衰减**：100个骰子去掉6点朝上的，联系指数衰减
- **COVID 疫情曲线**、城市人口、Gapminder GDP 数据
