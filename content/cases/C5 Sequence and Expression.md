# C5 Sequence and Expression

**文件：** case/sequence and expression.pdf（42 页）
**来源：** NCETM secondary assessment materials / 教学指南
**数学主题：** 数列/图形模式与代数表达式，从算术到代数的过渡
**适用年级：** Upper Elementary 到 Lower Secondary (9-14 岁)

## 活动 1：图形数列（Visual sequences）

用火柴棍/方块/点构成的图形规律：
- 第 1 个图形：4 根火柴（一个方框）
- 第 2 个图形：7 根（两个方框共享一边）
- 第 3 个图形：10 根
- 问：第 n 个图形用了多少根？

**步骤**：画前几个 → 数数量列表格 → 找公差 → 写通项 → 检验

## 活动 2：不同的"看法"产生不同的表达式

对同一个图形规律，不同方式"看"结构 → 不同但等价的表达式：
- 看法 1：**3n + 1**（每次加一组3根，最初有1根）
- 看法 2：**3(n-1) + 4**（第一个框4根，之后每增加一个框加3根）
- 看法 3：**4n - (n-1)**（n个框各4根，减去n-1个重叠的边）
- 然后证明等价（展开化简得到相同结果）

## 活动 3：从数值规律到通项公式

给定 2, 5, 8, 11, ...：
1. 判断公差（3）
2. 与 3n 数列（3,6,9,12...）比较
3. 用"shift"方法调整：3n - 1

## 活动 4：非线性数列

平方数列、三角数列、二次数列。用差分法（first/second difference）识别。

## 关键数学题目

- "Here are the first four terms: 5, 9, 13, 17. Find the nth term."
- "Two different students see this pattern differently. Student A sees groups of 3 with 1 extra. Student B sees groups of 4 with overlaps. Write expressions for each."
- "Which term in 3, 7, 11, 15, ... has value 99?"

## 设计特点

- 表征递进：火柴棍实物(enactive) → 表格(iconic) → 代数(symbolic)，体现 Bruner 三层
- 先个人思考 → 小组讨论不同"看法" → 全班共享
- 提供比较支架："与 2n 序列比较""与 3n 序列比较"
- 颜色标注图形中不同结构部分
- 跨文化比较：日本 TTP / 中国教材 / 英美文献

## 常见误概念

- 用递推思维而非封闭公式（只会说"每次加3"写不出 3n+1）
- 公差和第一项混淆
- 第 n 项从第 0 项还是第 1 项开始的模糊
- 同一规律怎么能有不同公式？
