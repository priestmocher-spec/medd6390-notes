# C8 Class Activities (WODB + PhET)

**文件：** example（gp）/class activity.pdf（31 页）
**来源：** Bush, Webb, Kress, Yang & Perkins (2018), 第六届国际 RME 会议论文
**数学主题：** 函数、代数表达式、比与比例、数的性质、几何分类
**适用年级：** Secondary

## PhET 模拟活动

### 1. Function Builder
- 拖拽输入卡片通过"函数机器"观察输出
- 第一屏（Patterns）：图片操作（旋转、翻转、变色、缩放）
- 后续：数字运算 → x 变量 → 方程/图表
- 可组合多个运算，观察顺序影响

### 2. Expression Exchange
- 硬币拖入工作区，同类可合并、不同类不能
- 系数自动出现帮助计数
- 可切换"硬币"和"x, y, z 变量"

### 3. Proportion Playground
- 颜料气球：调整蓝/黄数量观察混合色变化
- "能不能让蓝色气球比另一种少，但混合色还是偏蓝？"

## WODB (Which One Doesn't Belong?)

四个对象一组，**每个都有理由"不属于"**，没有唯一正确答案。促进数学推理和论证。

涵盖：数轴、百分比变化、比值、坐标几何线段、圆与直线位置关系

## 课程结构（所有模拟课共用）

1. **Open Play**：自由探索
2. **Whole Class Discussion**：分享观察
3. **Guided Exploration**：activity sheet 问题序列
4. **Formalization**：过渡到正式数学概念

## RME 三原则的体现

- **Activity Principle**：学生主动操作探索，没有唯一路径
- **Reality Principle**：混合颜料、收集硬币等可理解的情境
- **Level Principle**：从非正式到正式的渐进形式化

## PhET 设计原则（8 条）
1. Use implicit scaffolding
2. Make simulations highly interactive
3. Provide dynamic feedback
4. Use multiple, linked representations
5. Allow pedagogically powerful actions
6. Provide an interactive interface
7. Emphasize real world connections
8. Create a game-like environment

## 研究发现

- 6位教师，4所学校，2017-18学年
- 教师最常提及 Level principle，其次 Activity principle
- 学生几乎不提 Level principle
- **关键结论**：拥有情境和多重表征不等于渐进形式化自动发生
