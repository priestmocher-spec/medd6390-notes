# C1 Swan Card Sort

**文件：** case/Swan_card_sort.pdf（14 页）
**来源：** Malcolm Swan, MARS / Shell Centre, University of Nottingham
**数学主题：** 代数表达式、运算顺序、等价表达式、分配律
**适用年级：** 中学 (11-14 岁)

## 任务完整流程

### 1. 个人预评估（课前/课初 ~15min）
学生独立完成预评估任务：解释代数表达式含义，判断给定表达式是否等价。教师收回批阅，不给成绩，只识别学生的常见误概念。

### 2. 课堂卡片分类活动（~30min）
- 全班分成 2-3 人小组
- 每组拿到一套裁好的卡片（代数表达式）
- **Task 1**：将卡片分组，使同组表达式"always equal"（等价）。讨论并论证
- **Task 2**：将匹配好的组排列在大纸上，写下为什么等价的解释

### 3. 全班讨论
教师选择性展示某些小组的分组结果，引导讨论关键等价关系和常见错误

### 4. 后评估
再做一份类似任务检测进步

## 卡片上的数学内容

涉及的表达式（举例）：
- `2(n+1)` 与 `2n+2`（展开与因式分解的等价）
- `(n+1)^2` 与 `n^2+2n+1`
- `2n` 与 `n+n`
- `n^2` 与 `2n`（**常见混淆对**：不等价但学生常混淆，n=2 时恰好相等）
- `3n` 与 `n^3`（另一个常见混淆对）
- 空白卡片让学生自己写出等价表达式

## 脚手架设计

- 卡片中刻意包含"看起来像但不等价"的 distractors
- 提供 Hint Cards 给卡住的小组："试着代入 n=1, n=2, n=3 来检验"
- 空白卡片让学生自行补充等价形式

## 教师提问建议

- "你怎么确定这两个表达式**总是**相等的？只在某些值相等算不算？"
- "有没有可能找到一个 n 的值让这两个表达式不相等？"
- "你能用自己的话解释这个表达式的含义吗？"
- "这组里还缺少什么等价形式？"

## 常见误概念

- 混淆 2n 与 n^2（在 n=2 时恰好相等所以误判）
- 混淆 3n 与 n^3
- 展开括号错误：2(n+1) → 2n+1 而非 2n+2
- 认为字母只能代表一个特定的数而非变量
- 认为 n+2 和 2n 是同一回事

## 体现的理论

- 基于 formative assessment 理论（Black & Wiliam）
- 认知冲突（cognitive conflict）：通过小组讨论解决
- Card sort 迫使学生做判断和论证（justification），而非只做计算
- 引用 Swan (2005) Improving Learning in Mathematics
