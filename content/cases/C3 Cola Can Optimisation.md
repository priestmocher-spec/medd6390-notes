# C3 Cola Can Optimisation

**文件：** COLA-CAN-worksheet.pdf (1页) + Teacher-Notes-COLA-CAN.pdf (4页)
**来源：** AIMSSEC (African Institute for Mathematical Sciences Schools Enrichment Centre) Teacher Network
**数学主题：** 圆柱体体积与表面积、代数公式变换、约束条件下的最优化、列表法与图像法求极值
**适用年级：** Upper Secondary (Age 15+)；基础计算部分可用于 Lower Secondary (Grades 7-9)

## 学生工作单 (Worksheet)

### 核心问题

一个铝制可乐罐装 330ml 可乐。依次回答：

1. **正向计算：** 罐子直径 6cm，高度是多少？
2. **逆向计算：** 罐子高度 10cm，直径需要多大？
3. **比较：** 上述两个罐子哪个用铝更少？
4. **最优化：** 如果可以自由选择直径，什么尺寸的罐子用铝最少又能装 330ml？

### Help 部分 (scaffold, 6 步引导)

前提知识：1 毫升 = 1 立方厘米

1. 确认会计算圆柱体体积
2. 直径 6cm 时，需要知道如何计算底面积
3. 知道底面积后，可以算出满足 330ml 的高度
4. 从高度 10cm 出发，反推底面积、半径、直径
5. 思考表面积的组成部分（铝的用量），目标是让表面积尽可能小
6. 顶面和底面是圆形，但侧面积怎么算？（想象把标签从罐子上撕下来展开成长方形）

### Extension 拓展任务

用直径 1cm 的圆柱形木棒做一个 10cm x 10cm 的正方形框架，角落用 45 度斜切接合。

- 框架的体积和表面积是多少？
- 切割前木棒总长是多少？
- 找到一个连接体积、半径和外边长的公式

## 教师笔记 (Teacher Notes)

### 详细解法

**关键公式：**
- 圆柱体积：V = pi r^2 h = 330
- 由此得：h = 330 / (pi r^2)，或 r = sqrt(330 / (pi h))
- 表面积：S = 2 pi r^2 + 2 pi r h （两个圆 + 侧面矩形）

**具体计算：**
- 直径 6cm (r=3)：h = 330 / (pi x 9) = 11.67cm
- 高度 10cm：直径 = 6.48cm

### 系统列表法求最优解

教师笔记给出了一张完整的数值表（从半径 3 到半径 5 逐步递增）：

| 直径 (cm) | 半径 (cm) | 高度 (cm) | 表面积 (cm^2) |
|-----------|-----------|-----------|---------------|
| 6         | 3         | 11.67     | 276.55        |
| 7         | 3.5       | 8.57      | 265.54        |
| 7.3       | 3.65      | 7.88      | 264.53        |
| 7.4       | 3.7       | 7.67      | 264.40        |
| 7.5       | 3.75      | 7.47      | 264.36        |
| 7.6       | 3.8       | 7.27      | 264.41        |
| 8         | 4         | 6.57      | 265.53        |
| 9         | 4.5       | 5.19      | 273.90        |
| 10        | 5         | 4.20      | 289.08        |

**结论：** 直径约 7.5cm 时表面积最小，约 264.36 cm^2。图像呈非对称 U 形曲线。

### 诊断性评估 (Diagnostic Assessment)

**前置评估题：** 计算一个直径 6cm、高 20cm 的圆柱表面积（用 pi 表示）

- 选项：A. 120pi  B. 138pi  C. 180pi  D. 192pi
- 正确答案：**B (138pi)**
- 典型错误分析：
  - **选 A (120pi)：** 学生可能只算了 6 x 20 x pi，把给定数字直接相乘
  - **选 C：** 可能是猜的
  - **选 D (192pi)：** 学生估算了 3 x 20 x pi 得到 192，混淆了半径和直径

**评估流程（formative assessment 策略）：**
1. 把题写黑板上
2. 学生举手指表示答案（1指=A, 2指=B, 3指=C, 4指=D）
3. 让选不同答案的学生解释推理过程，不直接说对错
4. 全班再次投票，观察是否有人改变答案
5. 最后讲解正确答案

### 教学目标 (Learning Objectives)

- 练习圆柱体体积与表面积的计算
- 解决一个涉及两个变量的约束优化问题，寻找极值

### 通用能力 (Generic Competences)

1. 数学思维
2. 创造性地应用知识和技能
3. 解读和创建视觉图像来表征概念
4. 解读信息
5. 在多种情境中解决和解释问题

### 教学建议

- 确保学生知道 1ml = 1cm^3
- 用诊断题检查学生是否会算圆柱表面积和体积
- Showboard 展示板有助于课堂互动
- 让学生分组合作，分工计算不同直径对应的表面积
- 引导学生画图或制作电子表格

### Key Questions (引导问题)

- 如果圆柱体积 = 330，能找到什么公式连接 r 和 h？
- 能否把 r 写成公式的主项（subject）？
- 能否把 h 写成公式的主项？
- 体积 = 330 时，从直径出发能否算出 r、h 以及表面积？
- 能否列出数值表并画出表面积随直径变化的图像？

## 设计特点

- **真实情境驱动 (authentic context)：** 可乐罐是学生日常接触的物品，问题有真实的工程意义
- **分层 scaffold：** 工作单的 Help 部分提供 6 步渐进引导，学生可以按需使用
- **多重表征 (multiple representations)：** 公式 → 数值表 → 图像，三种表征方式互相验证
- **Cognitive demand 分析：** 起始问题（给定直径求高度）属于 procedures with connections；最优化问题属于 doing mathematics
- **Trial-and-improvement 策略：** 不用微积分也能通过系统列表找到近似最优解，降低了数学门槛
- **诊断性评估设计精巧：** 每个错误选项对应一种可识别的 misconception
- **Extension 任务：** 木框问题提供了进一步挑战，涉及不同几何体的体积和表面积
