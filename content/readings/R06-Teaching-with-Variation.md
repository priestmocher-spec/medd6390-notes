# 变式教学 (Teaching with Variation)

**来源**: teaching with variation.pdf (39 页)
**主题**: 变式理论在数学教学中的应用 — 中国与西方的比较

---

## 1. 核心论点与论证逻辑

本文的核心论点是：有意识地运用"变"与"不变"的对比，是帮助学生辨识数学结构的关键教学策略。作者论证了两条并行发展但殊途同归的理论线索：中国数学教育传统中的"变式教学"(teaching with variation) 和瑞典学习研究传统中的"变式理论"(variation theory)。

论证逻辑分三步。第一步，作者回溯了两个传统的各自发展脉络。中国的变式教学植根于数千年的考试文化和教学实践，顾泠沅等学者对其进行了系统化研究。瑞典的变式理论源于 Marton 和 Booth 的现象图析学 (phenomenography)，是关于学习如何发生的一般性理论。第二步，作者对比了两种理论的异同，发现它们在核心洞察上高度一致：学习的本质是辨识差异，而变式是制造差异的系统方法。第三步，作者论证了在实际课堂中如何运用这些原则来设计教学。

## 2. 关键概念定义与解释

### 变式理论 (Variation Theory)
Marton 等人提出的学习理论。核心观点：要学习某个东西，学习者必须辨识出那个东西的关键特征。而辨识的前提是经历变化。如果某个特征一直不变，学习者就不会注意到它。只有当它发生变化而其他东西保持不变时，这个特征才会变得"可见"。

### 学习的对象 (Object of Learning)
指教师希望学生学到的东西。包括两个方面：
- **直接对象 (Direct object)**: 具体的知识内容，如"理解分数加法"
- **间接对象 (Indirect object)**: 学习能力和思维品质，如"发展推理能力"

### 关键特征 / 关键面向 (Critical Features / Critical Aspects)
指学习对象中学生必须辨识到的关键属性。例如，学习"三角形面积"时，关键特征包括：底和高的关系、高必须与底垂直、高可以在三角形外部等。学生的困难通常来自没有辨识到某个关键特征。

### 变式的四种模式 (Patterns of Variation)

1. **对比 (Contrast)**: 通过展示"是什么"和"不是什么"的对比来突出关键特征。例如，同时展示三角形和非三角形，让学生辨识三角形的定义特征。
2. **分离 (Separation)**: 在保持其他特征不变的情况下，只变化一个特征，使该特征从背景中"分离"出来变得可见。例如，保持三角形的底不变，只改变高，让学生看到面积如何随高变化。
3. **概括 / 一般化 (Generalisation)**: 改变非关键特征而保持关键特征不变，让学生看到关键特征在不同情境中都成立。例如，用不同形状的三角形（等腰、直角、钝角）来计算面积，让学生看到面积公式对所有三角形都适用。
4. **融合 (Fusion)**: 同时变化多个关键特征，让学生在更复杂的情境中整合多个特征。例如，同时改变底和高，让学生综合考虑两者对面积的影响。

### 程序性变式 (Procedural Variation) — 中国传统
指在练习中系统地变化问题的形式和条件，帮助学生掌握不同的解题方法和策略。包括：
- **一题多解 (Multiple solutions to one problem)**: 用不同方法解同一题
- **一题多变 (Varying one problem)**: 改变一个问题的条件来生成新问题
- **多题一解 (One solution for multiple problems)**: 找出表面不同但本质相同的问题

### 概念性变式 (Conceptual Variation) — 中国传统
指通过变化概念的呈现方式来加深理解，包括：
- **正例 (Standard examples)**: 概念的标准呈现
- **变例 (Non-standard examples)**: 概念的非典型呈现
- **反例 (Counter-examples)**: 不属于该概念的例子
这三种变例的组合使用帮助学生建立完整的概念理解。

## 3. 具体例子与案例

### 平行四边形面积的变式教学
中国教师的做法：先教标准的平行四边形面积（底水平、高垂直）。然后呈现倾斜的平行四边形，让学生看到底和高的关系不取决于方向。再呈现底和高在图形外部的情况。通过这样的变式序列，学生的理解从特例推广到一般。

### 方程求解的程序性变式
原题：解 2x + 3 = 11。
变式一（改变系数）：解 3x + 5 = 20。
变式二（改变结构）：解 2(x + 3) = 22。
变式三（改变求解方向）：如果 2x + 3 = 11，那么 4x + 6 = ？
通过系统的变式，学生不仅掌握方程求解的步骤，还理解方程的结构。

### 分离模式的几何案例
教三角形面积时，教师展示一系列三角形：底都是 6 厘米，高分别是 2、3、4、5 厘米。因为只有高在变化（其他不变），学生能够清晰地"看到"高对面积的影响。这就是分离模式的应用。

### 对比模式在代数中的应用
同时呈现 3(x + 2) 和 3x + 2，让学生对比展开和不展开的区别。这种直接的对比帮助学生注意到括号的意义，这是很多学生忽略的关键特征。

### 中国课堂的"变式链"
一个典型的中国数学课可能围绕一个核心问题展开，通过不断变化条件来生成一系列相关问题。例如，从"一个长方形的长是 8、宽是 5，求面积"开始，变化为"面积是 40，长是 8，求宽"，再变化为"面积是 40，长比宽多 3，求长和宽"。每次变化都增加一个新的维度。

## 4. 框架/模型的完整描述

### 变式教学的设计框架

**第一步：确定学习对象和关键特征**
分析学生需要学什么，以及这个学习目标包含哪些关键特征。可以通过分析学生的典型错误和误解来确定关键特征。

**第二步：设计变式序列**
运用四种变式模式来设计系统的例子序列：
- 先用对比让学生注意到关键特征的存在
- 再用分离让学生理解每个关键特征的作用
- 然后用概括让学生看到概念的适用范围
- 最后用融合让学生综合运用多个特征

**第三步：安排教学顺序**
一般来说，从分离开始（一次只变一个特征），逐步走向融合（同时变多个特征）。但具体顺序要根据学习内容和学生情况来决定。

### 中国变式教学的双维框架 (顾泠沅)

**横轴 — 概念性变式**: 通过变化概念的呈现方式来加深理解
- 标准例子 → 变式例子 → 反例
- 目的：建构完整、灵活的概念理解

**纵轴 — 程序性变式**: 通过变化问题的形式和条件来发展技能
- 一题多解 → 一题多变 → 多题一解
- 目的：建立问题之间的联系，发展灵活的解题能力

两个维度应该在教学中同时运作，概念理解和程序技能相互支撑。

## 5. 对数学教学设计的具体启示

- **有意识地设计"变"与"不变"**: 教师在选择例题和练习时，应该有意识地控制哪些特征在变、哪些不变。随机选择例题是低效的。
- **分离模式是入门级策略**: 对于新概念，先用分离模式（一次只变一个东西）让学生注意到每个关键特征。这比一开始就呈现复杂的综合问题更有效。
- **用反例来纠正误解**: 当学生有特定的误解时，精心选择的反例比正面讲解更有效。例如，学生认为"对角线相等的四边形是正方形"，给他们看长方形就是有效的反例。
- **"一题多变"生成问题链**: 从一道核心问题出发，通过系统地改变条件来生成一系列问题，既节省设计时间，又能帮助学生看到问题之间的结构联系。
- **中国和西方的方法可以互补**: 中国的变式教学擅长通过精心设计的练习序列来发展深层理解和灵活技能。西方的变式理论提供了更系统的理论分析框架。两者结合能产生更好的教学效果。
